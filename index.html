🪟 Guía de Instalación para Windows

Si al abrir el instalador visualizas una pantalla azul de Microsoft Defender SmartScreen, es completamente normal en aplicaciones nuevas.

  1. Si ves el mensaje "Windows protegió su PC", haz clic en el enlace Más información.
  2. Aparecerá un nuevo botón. Haz clic en Ejecutar de todas formas para iniciar la instalación.

🍎 Guía de Instalación para macOS

macOS Recientes (Ventura, Sonoma, Sequoia y similares)

Interfaz moderna de Configuración del sistema.

Método 1 — Clic derecho (Siempre funciona)
  1. Descarga la app y ve a Finder → Aplicaciones.
  2. Haz Clic derecho sobre la app y selecciona Abrir.
  3. Confirma Abrir en la advertencia.
Método 2 — Configuración del sistema
  1. Ve a Configuración del sistema → Privacidad y seguridad.
  2. Desliza hasta el final (abajo).
  3. Pulsa Abrir de todos modos.

🐧 Guía de Instalación para Linux

Opción A: Archivo .AppImage (Portable)

Para ejecutar el archivo portable, necesitas darle permisos:

  1. Clic derecho en el archivo descargado.
  2. Ve a Propiedades → Pestaña Permisos.
  3. Marca la casilla "Permitir ejecutar el archivo como un programa".
  4. Doble clic para abrir.

O usa la terminal:

chmod +x *.AppImage && ./*.AppImage
Opción B: Paquete .deb (Debian/Ubuntu)

Instala usando el gestor de software o terminal:

sudo dpkg -i *.deb

Si hay errores de dependencias, ejecuta:

sudo apt-get install -f
